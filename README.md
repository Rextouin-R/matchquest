# MATCHQUEST bot | via TERMUX 

## Moduls by Shyzg

## Install 
```
pkg install git
```
```
pkg install nano
```
```
pkg install python
```
```
pkg install python3 python3-pip
```
```
git clone https://github.com/Rextouin-R/matchquest/
```
## Buka file
```
cd matchquest
```
```
python -m pip install -r requirements.txt
```
## Edit `queries.txt` lalu paste `query_id/user_id` kalian
## Save `CTRL+xy` (enter) 
```
nano queries.txt
```
## Run bot
```
python app.py
```
